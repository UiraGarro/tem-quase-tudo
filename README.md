#Escopo do Projeto

O presente projeto tem como objetivo o desenvolvimento de um site de e-commerce denominado “Tem Quase Tudo”, voltado à venda de produtos diversos em um ambiente virtual simples, funcional e acessível. O projeto tem caráter acadêmico e visa aplicar os conhecimentos adquiridos nas disciplinas de Desenvolvimento Web e Banco de Dados, integrando conceitos de front-end, back-end e persistência de dados.

O sistema permitirá que os usuários naveguem entre diferentes categorias de produtos, visualizem informações detalhadas de cada item, adicionem produtos ao carrinho e realizem um pedido. Além disso, será implementada uma área administrativa destinada ao gerenciamento de produtos, categorias e usuários, possibilitando o cadastro, edição e exclusão de registros.

O escopo do projeto abrange as seguintes funcionalidades principais:

Cadastro e login de usuários (clientes e administradores);

Exibição de produtos com informações como nome, descrição, preço e imagem;

Busca e filtragem de produtos por categoria ou palavra-chave;

Carrinho de compras e registro de pedidos;

Painel administrativo para gerenciamento de produtos e usuários;

Integração com um banco de dados relacional, responsável pelo armazenamento seguro das informações do sistema.

O desenvolvimento do site será realizado utilizando HTML, CSS e JavaScript no front-end, e PHP (ou outra linguagem escolhida) no back-end, com MySQL como sistema gerenciador de banco de dados. O projeto terá foco na usabilidade, organização do código e funcionamento básico das operações CRUD (Create, Read, Update, Delete).

Funcionalidades avançadas, como integração com meios de pagamento, controle de estoque em tempo real ou interface responsiva completa, estão fora do escopo inicial, podendo ser consideradas em versões futuras do sistema.
